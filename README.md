# Crypto Sentiment & Trader Performance Analysis
### Data Science Intern Assignment - Primetrade.ai

## 📊 Project Overview
This project analyzes how Bitcoin Market Sentiment (Fear/Greed Index) correlates with trader behavior and profitability. 

## 🚀 Key Insights
- **Market Sentiment:** Average PnL is significantly higher during "Extreme Fear" or "Extreme Greed" compared to Neutral days.
- **Trader Segmentation:** "Whales" outperform Retail traders by 4x during peak Greed cycles.
- **Behavioral Patterns:** Frequent traders capture consistent but smaller wins, whereas Infrequent traders capitalize on major sentiment shifts.

## 🛠️ Strategy Recommendations
- **The FOMO Filter:** Retail traders should scale down positions during "Extreme Greed" as data shows they often buy at the top.
- **Volatility Play:** The profit gap between Whales and Retail is smallest during "Extreme Fear," marking a potential entry point for smaller accounts.

## ⚙️ How to Run
1. Ensure the `data/` folder contains `sentiment.csv` and `trades.csv`.
2. Open `notebook/analysis.ipynb` in VS Code.
3. Select the Python kernel and click **Run All**.
